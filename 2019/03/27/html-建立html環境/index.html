html-建立html環境

基本環境架構

  • 定義網頁的格式
    因為html的時代悠久，有許多不同的html版本格式，因此若使用html5格式，需在第一行新增一行定義

  • 所有撰寫內容，都先用html包起來，裡面有裡面有兩個標籤： head、body

head:

放置網頁的相關資訊，例：網頁名稱、支援中文編碼模式、載入的css版本、載入的JS版本

該網頁的頁籤變更為「My first web」

body:

所有關於html的程式碼(網頁內容)，都放在body中
